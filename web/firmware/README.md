# Firmware Assets

This directory contains the browser-flasher image plus the published OTA binaries.

- `rsvp-m5-core2.bin` is the full browser-flasher image for the M5Stack Core2.
- `rsvp-m5-core2-ota.bin` is the Core2 OTA update binary.

For local exports, it is generated by:

```bash
python3 tools/export_web_firmware.py
```

The generated `.bin` files are ignored by Git. Manifests are tracked and have their version updated
by the export script.
